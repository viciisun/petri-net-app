Petri Net Layout 修改完成

✅ layoutService.js 重命名和功能增强完成

实现的功能：

1. 智能连接点分配

• 每个arc连接到node的不同attach point

• 根据节点相对位置自动选择最佳连接点

• 避免连接点重叠，自动分配到使用较少的连接点

2. 单向箭头

• 所有边都使用arrowclosed类型的箭头

• 箭头大小：20x20像素

• 边线样式：2px宽度，#333颜色

3. 节点显示优化

• 节点内部显示ID

• 节点下方显示name

• Place节点：圆形，ID在圆内，name在下方

• Transition节点：矩形，ID在矩形内，name在下方

4. 特殊标记显示

• Initial marking（有tokens的place）：绿色边框和渐变背景

• Final marking：红色边框和渐变背景

• 特殊标记的name标签也会有对应的颜色

5. 连接点颜色区分

• Target handles（输入）：蓝色 (#007bff)

• Source handles（输出）：绿色 (#28a745)
